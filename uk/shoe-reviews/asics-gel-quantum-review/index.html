Свідома золота середина

Технологія амортизації Gel від ASICS розташована в п'яті та передній частині стопи, надаючи Gel-Quantum помітно м'якший хід, ніж у спеціалізованих кросівках для важкої атлетики, як-от Metcon. Ця амортизація поєднана зі стабільною, а не м'якою й неструктурованою п'ятковою частиною, що не дає їм відчуватися хиткими під час бічних рухів на HIIT-занятті чи в колових тренуваннях.

Де вони кращі за кросівки для важкої атлетики

Для повної години змішаного кардіо й помірних силових вправ — типу заняття, що включає більшість групового фітнесу — додана амортизація Gel-Quantum дає реальну перевагу в комфорті над плоскоп'ятковими кросівками для важкої атлетики. Відгуки покупців послідовно описують їх як поблажливіші протягом усього заняття, особливо для стрибків і бічних рухів, поширених у коловому тренуванні.

Універсал, а не спеціаліст Плата за цю універсальність — нижча стеля під максимальним навантаженням. Якщо ваші тренування здебільшого важка робота зі штангою, жорсткіші, плоскоп'яткові кросівки на кшталт Nike Metcon дадуть стабільнішу базу.

Посадка

Доступні у стандартній і широкій ширині, що корисно для ширшої передньої частини стопи, поширеної разом із плоскостопістям, хоча діапазон не такий широкий, як у спеціалізованого взуття для ходьби на кшталт ліній New Balance.

Де вони поступаються

  • Не для максимальних піднять. Амортизація стискається під важким навантаженням зі штангою швидше, ніж у плоскоп'яткових кросівок для важкої атлетики.
  • Довговічність під навантаженням. Не створені спеціально для витримування повторюваних важких падінь штанги.
  • Золота середина. Не перевершують спеціально ні в кардіо, ні в силових тренуваннях так, як спеціалізовані кросівки для кожного з них.

Кому підходять ці кросівки

Гарний вибір

Будь-кому, хто відвідує групові фітнес-заняття або поєднує кардіо з помірними силовими вправами і хоче одні універсальні кросівки для залу.

Шукайте інше, якщо

Ви серйозний пауерліфтер, якому потрібна максимальна стабільність під важким навантаженням зі штангою — перегляньте натомість наш огляд Nike Metcon.

Потрібна більша стабільність для важчих піднять? Див. наше порівняння Nike Metcon та ASICS Gel-Quantum.

Підсумок

Для плоскостопості в умовах змішаних тренувань — трохи кардіо, трохи силових, нічого максимального — баланс амортизації й стабільності Gel-Quantum є розумною золотою серединою.

Часті запитання

Чи годиться ASICS Gel-Quantum для силових тренувань?

Придатні для помірних силових тренувань, але амортизованіша підошва менш стабільна під максимальним навантаженням, ніж плоскоп'ятові кросівки для важкої атлетики, як-от Nike Metcon.

Кому найкраще підходить Gel-Quantum?

Тим, хто відвідує групові фітнес-заняття, HIIT-тренування чи поєднує кардіо з помірними силовими вправами, де комфорт протягом цілої години важливіший за максимальну стабільність під штангою.
