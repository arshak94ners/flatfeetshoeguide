Чому плоска, жорстка п'ята важлива під навантаженням

Більшість тренувальних кросівок пом'якшують п'яту для комфорту. Metcon робить навпаки навмисно — жорстка, близька до землі п'яткова пластина мінімізує стискання, коли навантажена штанга опускається. Для плоского, гнучкого склепіння, якому й так доводиться більше працювати для стабілізації стійки, ніж нейтральному склепінню, усунення цього додаткового шару м'якої підошви дає пряміший, передбачуваніший зворотний зв'язок із землею.

Де цей компроміс коштує вам

Та сама жорсткість, яка допомагає під штангою, робить Metcon поганим вибором для будь-чого, що нагадує тривале кардіо. Відгуки покупців послідовно відзначають дискомфорт на довших пробіжках чи тривалих стрибках зі скакалкою — просто менше амортизації для поглинання повторюваного удару. Це кросівки для важкої атлетики, вбрані в одяг кросфіт-тренувального взуття, а не справді універсальні кросівки для залу на всі випадки.

Підбирайте кросівки під тренування, а не просто під зал Якщо ваші заняття здебільшого силові вправи зі штангою, стабільність Metcon — реальна перевага. Якщо це переважно HIIT-заняття чи змішане кардіо, амортизованіший варіант на кшталт ASICS Gel-Quantum імовірно відчуватиметься краще протягом цілої години.

Довговічність

Nike створює лінію Metcon, щоб витримувати підйоми по канату, падіння штанги та загальні знущання в залі, і відгуки покупців послідовно оцінюють її довговічність значно вище за типові тренувальні кросівки. Ця міцність коштує м'якшого, поблажливішого ходу, який дав би амортизований тренувальний кросівок.

Де вони поступаються

  • Не для кардіо. Жорсткий хід некомфортний для тривалого бігу чи стрибків.
  • Ціна. Преміальна ціна порівняно із загальними тренувальними кросівками.
  • Обмежена амортизація. За задумом, дає менше гасіння ударів, ніж спеціалізовані тренувальні кросівки.

Підсумок

Metcon — це кросівки для спеціаліста, а не універсала. Для плоскостопості, що займається серйозною важкою атлетикою, компроміс зі стабільністю того вартий; для змішаного кардіо й силових — це неправильний інструмент.

Часті запитання

Чи годяться Nike Metcon для бігу?

Не дуже. Жорстка, майже плоска п'ята створена для стабільності під час важкої атлетики, а не для амортизації під час бігу, і зазвичай відчувається жорстко на довших пробіжках чи кардіо-заняттях.

Чому плоска, жорстка п'ята допомагає при плоскостопості під час силових тренувань?

Плоскому, рухливому склепінню й так доводиться більше працювати, щоб стабілізувати вашу стійку під навантаженням. Жорстка, мінімально амортизована п'ята дає пряміший, передбачуваніший зв'язок із землею, зменшуючи нестабільність, яку додала б м'яка, товста підошва.
